# restaurante-sistema1
Proyecto 4: Sistema de Gestión de Restaurante

🌸 Sistema de Gestión - Restaurante Sakura
¡Bienvenido al proyecto Sakura! Este es un sistema de administración web desarrollado para la gestión integral de un restaurante japonés. Permite manejar desde la visualización del menú para clientes hasta el control de staff y seguridad de acceso.

🚀 Funcionalidades
Página de Inicio Dinámica: Menú autogenerado desde la base de datos MySQL.

Sistema de Login: Acceso seguro para empleados con validación de credenciales.

Panel Administrativo (Dashboard): Interfaz centralizada para el control del negocio.

Gestión de Staff (CRUD): Visualización, edición y eliminación de usuarios registrados.

Seguridad por Roles: Restricción de funciones según el puesto (Jefe/Mesero).

🛠️ Tecnologías Utilizadas
Backend: PHP 8.x

Base de Datos: MySQL (MariaDB)

Frontend: HTML5, CSS3 (Diseño Dark Mode)

Servidor Local: XAMPP / Laragon

📋 Requisitos e Instalación
Clonar o descargar el repositorio en tu carpeta htdocs de XAMPP.

Importar la base de datos:

Abre phpMyAdmin.

Crea una base de datos llamada restaurante_db.

Importa el archivo .sql (si lo tienes) o crea las tablas usuarios y platillos.

Configurar la conexión:

Revisa el archivo includes/database.php y asegúrate de que el usuario y contraseña de MySQL sean los correctos.

Ejecutar:

Abre tu navegador y ve a localhost/restjap1/index.HTML..
